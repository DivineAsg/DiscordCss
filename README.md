# Discord Custom CSS (Vencord / BetterDiscord)

Este repositório reúne um conjunto de customizações em CSS para o Discord, com foco em ajustes visuais e organização da interface.

As modificações alteram principalmente o comportamento visual do cliente, com mudanças na disposição, transparência e estilo geral da UI.

## O que foi modificado

- Ajustes na interface para um visual mais limpo e compacto  
- Uso de transparência em diferentes áreas do aplicativo  
- Alterações no formato padrão da interface (bordas e espaçamentos)  
- Mudanças no comportamento de elementos como avatares e componentes da UI  
- Personalização de fonte e tipografia  
- Remoção ou ocultação de elementos considerados desnecessários  

## Observações

Este tipo de customização depende fortemente da estrutura interna do Discord, então atualizações do cliente podem quebrar partes do CSS.

## Compatibilidade

- Vencord  
- BetterDiscord (parcial)  
- Outros clientes que suportem injeção de CSS
